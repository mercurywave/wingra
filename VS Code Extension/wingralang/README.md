# wingralang README

This is the README for the Wingra programming language extension for VSCode

## Features

Adds support for editing and running .wng and .wingraProj files and projects.

## Requirements

Requires Windows

## Extension Settings

No settings currently

## Known Issues

None

## Release Notes
